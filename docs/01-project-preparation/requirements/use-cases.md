# Use Cases - Smart Civic Registry

## UC-01: Person anlegen
**Akteur**: Sachbearbeiter  
**Beschreibung**: Erfassung einer neuen natürlichen Person im System  
**Vorbedingung**: Sachbearbeiter ist eingeloggt mit Rolle OFFICER oder ADMIN  
**Nachbedingung**: Personendatensatz ist gespeichert mit Audit-Trail

**Hauptszenario**:
1. System zeigt Personen-Suchmaske
2. Sachbearbeiter sucht nach bestehender Person
3. System zeigt keine Treffer
4. Sachbearbeiter wählt "Neue Person anlegen"
5. System zeigt Personenerfassungsformular
6. Sachbearbeiter füllt Pflichtfelder aus...
